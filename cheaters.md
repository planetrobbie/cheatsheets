#### General
Action | Function
-- | --
f {First letters}| Fast Switcher by name
f {1..9}| Fast Switcher by number
Command-I | Change Contrast

#### Moving 
Action | Function
-- | --
xCMDxSHIFTH | Home
t | Table of content
, | Previous Section
. | Next Section
j | Up
k | Down
gg | Top
G | Bottom
h | Previous Cheatsheet
l | Next Cheatsheet

#### Search

Action | Function
-- | --
f? {query}| DuckDuckGo
f?g {query}| Google
?gh {query}| GitHub
?so {query}| Stack Overflow

#### Markdown
Type (without space) | For
-- | --
x CMD | Command
x SHIFT | Shift
x CTRL | Control
x OPT | Option

Put Markdown in `~/in/code/cheaters-master/cheaters/cheatsheets`
