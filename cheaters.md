# Cheaters

#### General

Action | Function
-- | --
f {First letters} | Fast Switcher by name
f {1..9} | Fast Switcher by number
Command-I | Change Contrast

#### Navigate

Action | Function
-- | --
xCMDxSHIFTH | Home
t | Table of content
, | Previous Section
. | Next Section
j | Up
k | Down
gg | Top
G | Bottom
h | Previous Cheatsheet
l | Next Cheatsheet

#### Search

Action | Function
-- | --
f? {query} | DuckDuckGo
f?g {query} | Google
?gh {query} | GitHub
?so {query} | Stack Overflow

#### Markdown

Type (without space) | For
-- | --
x CMD | xCMD
x SHIFT | xSHIFT
x CTRL | xCTRL
x OPT | xOPT

Put Markdown cheatsheets in `~/in/code/cheaters-master/cheaters/cheatsheets`
